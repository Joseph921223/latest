Employee Monitoring System AC Documentation

Introduction

This document is intended to give clear views of the employee monitoring system. Highlighting the architectures, design process, user interaction of both hardware and software part, this document directs the user on the usage of the system and how it can best be used to get accurate results from it.

System Overview

This system test if employees are active enough to start working on that particular day, it also uses an android application to activate the sensors to read in given time, the user/controller control the system from the android application.

Hardware system

This system will be built with the wooded container for constant temperature control since wood don’t trap temperature like metal do, it will consist of 3 slots, Temperature sensor slot, pulse reading slot and alcohol level measuring slot.

  • Heartbeat sensor - For the heartbeat sensor will use the pulse reader to sense the pulse in the finger. It will be in the deeper most part of the container to reduce light interference to it
  • Temperature sensor - Temperature sensor will be used to measure temperature from the finger but also flexible enough to be put to measure it in the mouth.
  • Alcohol sensor - It will be covered with an inner plastic cover for easy and direct breath capture All the system components will have indicator lights to give status, red for when reading and green for done

System Architecture

With the on-going growth of technology, more and more machineries are introduced into manufacturing industries. Machineries faster and even sharper for the benefit of the company, looking at profit side of it. Some of these machineries use technology more advanced like artificial intelligence but some of these machines still need human operations, which then introduce human errors. Human error both caused by human limitations andPage 7 of 12 reckless behavior which include operation these machines under the influence of alcohol and forcing your body and mind to focus even when not fit, in cases of illness. As the results of these human errors more and more injuries on duty are reported.

This system is designed to bridge that gap by ensuring that only employee who are mentally and physically fit are the one set to operate those sensitive machineries on daily bases. This system will test for alcohol level in the body, heartbeat rate and body temperature. Based on the collective results obtained from the system, the employer will know which individuals are fit and focused to operate these machines thus human errors out of recklessness will be reduced. This system is the solution because most companies don’t have such system and have to relay of the sign they can spot from employees by sight, or maybe by smell of alcohol in close contact, but then using this system accurate results will be produced.
